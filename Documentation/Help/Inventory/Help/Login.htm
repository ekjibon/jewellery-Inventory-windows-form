Inventory system has two user roles; Super user is a predefined user and super user will have all the rights
to the system. All other user will be considered as non admin users.
This system has privilege to create any number of non super users only.

Non super user is restricted to following access:
  • Managing financial year
  • Managing configuration
  • Master Maintenance
  • information settings
  • DB backup utility
  • User Management settings (Creating and Editing users)

When we run the application the first window is a login form which has already system user known as Superuser.
This is for authentication purpose. Then the master form is open firstly it ask for setting the financial year. (Next time we use only FY for Financial Year)
