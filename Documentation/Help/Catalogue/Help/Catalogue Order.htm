Catalogue Order:

This form allows us to generate “Marketing Order” or “Customer Order” or “Catalogue Order” form, all are for the same.
  • Enter appropriate Quotation No and Customer Name.
  • If we know the Jewel No, we can directly enter and then press Enter key. OR Put your cursor on Jewel No textbox and scan barcode tag
    with barcode machine. It will read the barcode having Jewel No and if data is available it will display all the jewel information on the grid.
  • Even we can use Design Code or Style No to search jewels.
  • Once we are done with the search part, we can Select and Unselect jewels in the grid as per our choice.
  • Click on “Process” button to generate the order form.
