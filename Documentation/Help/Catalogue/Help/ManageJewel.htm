Manage Jewels:
  • This form allows saving alloy jewels information.
  • Jewel no is auto generated from database.
  • Give appropriate style no, for example for Necklaces use NK01.
  • Click upload button to select image for the jewel.
  • Enter all the inputs and click on save button.


  • Click on “Print Sticker” button to print the jewel sticker as shown below.



  • Use right hand side report, to print the jewel barcode tag


    • Print Tag for All Jewels:

    This form allows us to generate & print barcode for selected jewels.

    • Select jewels from the grid using checkboxes, maximum 10 jewels can be selected.
    • Click “Generate Barcode” button to generate barcode for selected jewels.
    • Click print to print the barcode tags.
    • If there are more than 10 records that will go to next page, you can use “Select Page No:” to access more records.
    • Click “Select All” button to select all the jewels (max. 10) on selected page.

    • Click “Generate Barcode” button to generate barcode.

    • Click on print to print the barcode tags.
